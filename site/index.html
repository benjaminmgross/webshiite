welcome to my website

I've put together a repository of my experiences -- both personal and professional -- to serve as a reference.

This has almost nothing to do with a belief that I'm "the expert" on a given topic, but rather I tend to forget things pretty quickly, and explain / document things pretty well. This means I can use this site to refer to things I've done previously and other people can use my organized meanderings, should they find overlap in their own personal spheres of interest.

In general, the topics covered on this website, which mirrors the navbar structure in the upper right of this website is as follows:

  • Professional background and current projects
  • Experiences and personal notes around programmatic development, specifically:
    • Shell scripting (for those unfamiliar, that's what you get when you open Terminal on a Mac)
    • Python
    • New explorations into JavaScrapt, HTML, and css
    • Emacs (or Aquamacs for Mac OS X)
  • Blog posts about personal experiences
  • Passions including, but not limited to:
    • Interpersonal Communication
    • Meditation
    • Personal Fitness
    • Animal Awareness
    • Dog Training
    • Computational Finance
  • Current projects
